# Create Pull Request Command

## Instructions
1. Create new branch: `feature/issue-[number]-[short-name]`
2. Commit all changes with descriptive messages
3. Create pull request with:
   - Title: "[Issue #] Feature description"
   - Body: Link to issue PRD and summary of changes
   - Request review from CodeRabbit and other AI reviewers

## PR Template
```
## Changes Made
[Brief bullet points of what was implemented]

## Issue PRD Reference
Link: docs/new-issues/issue-[number]-[name]-prd.md

## Testing
- [ ] All existing tests pass
- [ ] New tests added and passing
- [ ] Manual testing completed

## Checklist
- [ ] Follows code patterns specified in PRD
- [ ] Meets all acceptance criteria
- [ ] Documentation updated
- [ ] No breaking changes introduced
```
